## Respuestas del Taller:
1. **AndroidManifest.xml:** Es el archivo esencial que describe información del paquete, componentes (actividades, servicios) y permisos al sistema Android.
2. **activity_main.xml:** Es el archivo de diseño (layout) donde se define la interfaz de usuario usando etiquetas XML.
3. **MainActivity.kt:** Es la clase principal en Kotlin que contiene la lógica de la pantalla, como el control del contador y los eventos de clic.
4. **Emulador Android:** Es un dispositivo virtual (AVD) que simula un teléfono Android en la computadora para probar aplicaciones sin hardware físico.
